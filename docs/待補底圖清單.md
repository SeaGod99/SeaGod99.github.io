# 待補地圖底圖清單（2026-06-11）

> 地圖ID統一修正計畫產物。Cowork 沙箱無法下載 XIVAPI 圖片資產，**請在本機 repo 根目錄執行**：
>
> ```
> node scripts/download-maps.mjs
> ```
>
> 腳本已改為預設只抓野外/主城/居住區（field/city/housing），會自動跳過已存在的檔案，可重複執行。
> 副本/特殊區域（dungeon/instance，共 134 張）依 2026-06-10 決議**不下載**，maps.json 留 `image.url` 欄位，需要時再跑 `node scripts/download-maps.mjs --all`。

預設與 `--all` 之間的折衷：用 `--id` 單獨補這張，不必多抓上百張副本圖：

```
node scripts/download-maps.mjs --id 584
```
