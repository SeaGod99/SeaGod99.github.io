# Universalis API

### 備註
- 名稱解析使用精確匹配；外站不可用時會回退本地資料庫。
- 價格採用 Universalis 聚合端點；利潤分析包含最低與平均價格的對比。
---

## 🎯 前端功能（Web UI）

### 1. 物品查詢（快速查詢名稱 & 價格）
- **位置**：「製造分析」卡片上方的搜尋按鈕
- **功能**：
  - 輸入物品名稱與伺服器
  - 按「🔍 查詢名稱 & 價格」按鈕
  - 自動填入 ID 並顯示 NQ/HQ 最低與平均價格
- **優勢**：快速確認物品名稱與當前市場價格

### 2. 製造分析（完整分析）
- **位置**：主要的「製造分析（價格 + 成本 + 利潤）」卡片
- **輸入**：物品名稱、伺服器、製造數量
- **輸出**：
  - 物品資訊（名稱、ID、來源）
  - 市場價格（NQ/HQ 最低與平均）
  - 製造成本（總成本、單位成本、職業等級、詳細材料）
  - 利潤分析（最低掛單/平均售價的利潤、可獲利判定）
- **適用**：決策是否值得製造與販售該物品
