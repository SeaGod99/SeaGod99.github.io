# FFXIV 市場查詢工具

即時查詢最終幻想 XIV 市場板價格和製作成本分析工具

---

## ✨ 主要功能

### 🔍 物品搜尋
- **多語言支持**：支援繁體中文與英文搜尋
- **即時搜尋**：輸入物品名稱即可快速查詢
- **職業篩選**：依製作職業（鍛鐵、鑄甲、雕金等）快速過濾搜尋結果
- **詳細資訊**：顯示物品等級、製作難度（RLV）、職業分類等

### 💰 市場價格查詢
- **即時價格**：查詢 NQ/HQ 的最低與平均市場價格
- **多伺服器支持**：支援所有 FFXIV 伺服器與資料中心
- **價格更新時間**：顯示市場數據的最後更新時間（相對時間格式）
- **自動重新整理**：價格資訊自動載入，無需手動操作

### 🔨 合成表與製作成本
- **完整合成表**：顯示製作配方的所有材料需求
- **成本計算**：自動計算所有材料的市場價格與總成本
- **材料價格追蹤**：每個材料顯示單價、小計與市場更新時間
- **子配方支持**：顯示可製作材料的子配方成本與省錢分析
- **利潤分析**：比較製作成本與市場售價，計算利潤空間
- **批量製作**：支援自訂製作數量，自動計算總成本

### 🌐 多語言支持
- **繁體中文**：繁中物品名稱
- **English**：英文物品名稱

---

## 🎯 使用流程

### 1. 選擇伺服器
1. 選擇你的資料中心（Data Center）
2. 選擇你的伺服器（World）
3. 選擇搜尋語言（繁中/英文）
4. 點擊「✓ 確認伺服器」

### 2. 搜尋物品
1. 在「🔍 物品搜尋」面板輸入物品名稱
2. 點擊「搜尋」或按 Enter 鍵
3. 從搜尋結果中選擇目標物品

### 3. 查看資訊
- **物品信息**：查看物品名稱、ID、伺服器資訊
- **市場價格**：查看 NQ/HQ 最低與平均價格，以及更新時間
- **合成表**：查看製作配方、職業、等級、難度等資訊
- **材料成本**：查看每個材料的市場價格與更新時間

### 4. 計算製作成本
1. 輸入製作數量
2. 點擊「計算成本」
3. 查看總成本、單位成本與成品市場價格對比

## 📊 資料來源

- **XIVAPI**：物品基礎資訊與配方資料
- **Universalis API**：即時市場價格資料
- **TNZE API**：繁體中文物品名稱資料

---

## 💡 使用提示

- **選對語言**：搜尋前記得選擇正確的語言，中文物品請用繁中搜尋
- **查看更新時間**：注意市場價格的更新時間，過舊的資料可能不準確
- **比較子配方**：材料如果可製作，會顯示子配方成本比較，幫助你決定是買還是自己做
- **批量計算**：大量製作時記得調整數量，系統會自動計算總材料需求
