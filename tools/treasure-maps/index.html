🗺️ 藏寶圖採集點查詢

陳舊的地圖 G1～G17 各等級的挖寶位置，對照解讀後的藏寶圖找出挖掘地點

選擇藏寶圖等級

在地圖上解讀（使用）藏寶圖後，會顯示一張挖寶地點的碎圖；選等級→選地區後，和右側碎圖比對，點相同的那張，左側地圖的紅釘就是挖掘點。
標示 組隊寶物庫 的等級，開啟寶箱後有機率出現傳送門（宇宙／樂園之門），建議 8 人同行。
🚩 標旗 複製的是 /coord X Y 地區 指令：搭配 ChatCoordinates 類插件貼入聊天即自動於地圖標旗；一般環境（主機／未裝插件）貼上仍是可讀座標，遊戲原生無法由文字直接產生地圖旗標。
